# Simulador de Processo Seletivo

## 📌 Sobre o projeto

Este projeto simula um processo seletivo para avaliação de candidatos utilizando Java.

O objetivo é representar um cenário simples de recrutamento, aplicando conceitos de Programação Orientada a Objetos como encapsulamento, herança, polimorfismo e interfaces.

## 🚀 Funcionalidades

- Cadastro de candidatos.
- Definição da nota pelo recrutador.
- Avaliação dos candidatos com base na nota atribuída.
- Exibição das informações do candidato.

## 🛠️ Tecnologias utilizadas

- Java
- Git
- GitHub

## 📚 Conceitos aplicados

- Programação Orientada a Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Interfaces

## ▶️ Como executar

1. Clone este repositório.
2. Abra o projeto em uma IDE Java (como IntelliJ IDEA ou Eclipse).
3. Execute a classe `Main`.
4. Utilize o menu do sistema para interagir com o processo seletivo.
